Tutti i modi per centrare un elemento

1. margin: 0 auto

Per elementi block con larghezza definita

margin: 0 auto
.box { width: 200px; margin: 0 auto; }

2. position: absolute + transform

Centra perfettamente indipendentemente dalla dimensione

absolute + transform
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

3. position: absolute + margin (dimensione fissa)

Funziona solo se conosci larghezza e altezza

absolute + margin
.box {
  width: 200px;
  height: 100px;
  position: absolute;
  top: 50%;
  left: 50%;
  margin-top: -50px; /* met√† altezza */
  margin-left: -100px; /* met√† larghezza */
}

4. position: absolute con tutti i lati a 0

Funziona con dimensione fissa

absolute con tutti i lati a 0
.box {
  width: 200px;
  height: 100px;
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  margin: auto;
}
