🎯 CENTRARE IN CSS

Tutti i modi per centrare un elemento

1. margin: 0 auto

Per elementi block con larghezza definita

margin: 0 auto
.box { width: 200px; margin: 0 auto; }

2. position: absolute + transform

Centra perfettamente indipendentemente dalla dimensione

absolute + transform
.box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

📊 Confronto diretto

margin: 0 auto

orizzontale

✓ Centra solo orizzontalmente
✗ Non centra verticalmente

transform 50%

perfetto!

✓ Centra sempre
✓ Non serve conoscere dimensioni

✨ LA FORMULA MAGICA (la migliore in assoluto)

position: absolute;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);

Vantaggi: Funziona sempre, non serve conoscere le dimensioni, centra perfettamente sia orizzontalmente che verticalmente!

📝 Quando usare cosa:

  • margin: 0 auto → solo per centrare orizzontalmente elementi block
  • transform: translate(-50%, -50%) → per centrare SEMPRE e OVUNQUE (il migliore!)
